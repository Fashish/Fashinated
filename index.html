My Course Notes Thus far with some Style!

  • A selector in CSS refers to a word in the CSS file that identifies which element the rules within the selector should apply to.
  • The CSS Reference is a comprehensive document that lists all standard CSS properties, pseudo-classes, pseudo-elements, @-rules, units, and selectors all in alphabetical order as well as showing how to use the basic syntax of each CSS rule.
  • Another great resource for CSS documentation is this article which shows how CSS selectors work.
  • Some CSS vocabulary:
    • CSS: Cascading Style Sheet
    • Rule: a line of CSS code describing the value that a certain attribute should use.
    • Property: The property you want to change.
    • Value: The value that you want to assign to the attribute
    • Selector: The name that you use in order to target the elements that are assigned to a class or id attribute in the HTML.
    • Class: A class refers to a group of elements that can be styled together.
    • ID: ID's are unique identifiers that uniquely identify an element in HTML.
  • This way of using styles is typically bad practice and should be avoided.
  • Styles can either be incorporated through a CSS file separate to the html one with a link tag added to the head tag in the html, or added within the style tag at the top of the html.
  • Setting the box-sizing attribute in the CSS to border-box makes the calculation of the element size to include both the border and padding.
  • Adding the rule display: flex; to the appropriate div will override their behaviour of appearing as blocks that take up the entire screen and let them appear next to each other.
